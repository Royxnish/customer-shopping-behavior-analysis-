# 🛍️ Customer Shopping Behavior Analysis

> *What makes a customer come back? What makes them spend more? This project digs into those questions using real retail transaction data.*

---

## What This Project Is About

I analyzed a dataset of **3,900 customer transactions** from a retail company to understand what's actually driving their sales — and what isn't. The management team had a feeling that things were shifting: different demographics buying differently, some products outperforming others for no obvious reason, and loyalty programs that weren't quite hitting the mark.

So I went through the data end-to-end — cleaned it in Python, queried it in SQL, visualized it in Power BI, and wrote up what it all means for the business.

The core question I was trying to answer:

> *"How can the company use what it already knows about its customers to sell smarter, engage better, and keep people coming back?"*

---

## What I Found (The Short Version)

A few things genuinely surprised me:

- **Subscribed customers don't actually spend more.** The average purchase for subscribers ($59.49) is almost identical to non-subscribers ($59.87). The subscription program needs a rethink.
- **80% of customers are already loyal** — they've made 11+ purchases. That sounds great, but it also means almost nobody new is coming in. Acquisition is a real gap.
- **The 55+ age group generates the most revenue** ($69,590 — nearly 30% of total). Meanwhile, the 18–24 group is the smallest contributor. Big opportunity there.
- **Male customers dominate** — 67.7% of revenue. The female segment is underserved and likely undermarketed to.
- **Gloves, Sandals, and Boots** are the highest-rated products. They deserve more spotlight in campaigns.
- **43% of transactions used a discount**, but many of those customers still spent above average. Discounts aren't just attracting bargain hunters.

## The SQL Questions I Explored

These are the 10 business questions I answered with SQL:

| # | Question |
|---|----------|
| Q1 | What is the total revenue generated by male vs. female customers? |
| Q2 | Which customers used a discount but still spent above average? |
| Q3 | Which are the top 5 products by average review rating? |
| Q4 | Does shipping type (Standard vs Express) affect average spend? |
| Q5 | Do subscribed customers actually spend more? |
| Q6 | Which products have the highest discount usage rate? |
| Q7 | How do customers break down into New, Returning, and Loyal segments? |
| Q8 | What are the top 3 products within each category by purchase volume? |

## How to Run This Locally

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/customer-shopping-behavior-analysis.git
cd customer-shopping-behavior-analysis
```

### 2. Set up your environment variables
```bash
cp .env.example .env
```
Then open `.env` and fill in your actual PostgreSQL credentials:
```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=your_database_name
DB_USER=your_username
DB_PASSWORD=your_password
```

### 3. Install Python dependencies
```bash
pip install pandas psycopg2-binary sqlalchemy python-dotenv
```

### 4. Run the notebook
Open `python/data_cleaning.ipynb` in Jupyter and run all cells. This will clean the dataset and load it into your PostgreSQL database.

### 5. Run the SQL queries
Open `sql/customer_behaviour_queries.sql` in pgAdmin or any SQL client connected to your database and run the queries.

### 6. Open the dashboard
Open `powerbi/Customer_behaviour_dashboard.pbix` in Power BI Desktop.

## Key Recommendations

Based on the analysis, here's what I'd tell the business team:

1. **Go after female customers more deliberately** — targeted campaigns, curated collections, referral programs. The revenue gap is 35% and mostly unexplained by product preference alone.
2. **Make the subscription worth it** — add real perks like free shipping, early access, or loyalty tiers. Right now it's not moving the needle on spend.
3. **Don't ignore the 18–24 segment** — social media, student discounts, trend-led capsule collections. They're the smallest group today but the most valuable long-term.
4. **Be smarter with discounts** — use them surgically on slow-moving SKUs, not as a blanket strategy. Many high-intent buyers don't need them.
5. **Plan inventory around Spring and Winter** — those are peak seasons. Use Fall to run loyalty campaigns before the Winter rush.
6. **Put Gloves, Sandals, and Boots front and center** — highest customer satisfaction scores. Let the reviews do the marketing.
